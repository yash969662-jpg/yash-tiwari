---

### 2. 🌤️ Weather Dashboard
A beautiful weather application with real-time data and forecasts.

**Files:**
- `weather.html` - Main weather dashboard
- `weather-style.css` - Styling
- `weather-script.js` - Functionality

**Features:**
- Real-time weather for any city worldwide
- Current weather conditions with detailed metrics
- 5-day weather forecast
- 24-hour hourly forecast
- Beautiful gradient UI with animations
- Responsive mobile-friendly design
- Free API (no API key required)
- Quick search for popular cities

**How to Use:**
1. Open `weather.html` in your browser
2. Search for any city
3. View current weather and forecasts

---

## Technologies Used

- **CSS3** - Modern styling, gradients, animations, flexbox, grid
- **JavaScript ES6+** - DOM manipulation, async/await, fetch API
- **Font Awesome** - Icons
- **Open-Meteo API** - Free weather data
